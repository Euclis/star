本项目在Firefox版本: 56.0.1 (32 位)、Chrome版本 62.0.3202.94(32 位)下测试
    
1、使用了rem，可根据屏幕宽度改变比例，响应式布局

2、滑动效果使用了jQuery全屏滚动插件fullPage.js

3、单屏特效使用了css3

4、注册框使用了ajax技术
